# Arquivo

Esta pasta contém projetos concluídos, metas alcançadas, e outros materiais que não são mais ativos mas podem ser úteis para referência futura.

## Como Usar:

1. Mova projetos concluídos de `projects/` para cá
2. Arquive notas antigas que não são mais relevantes
3. Mantenha uma estrutura similar à pasta original
4. Adicione data de conclusão aos arquivos

## Organização:

```
archive/
├── 2026/
│   ├── Q1/
│   ├── Q2/
│   └── projects/

```
