# Áreas da Vida

Esta pasta contém suas diferentes áreas de responsabilidade e interesse na vida. Áreas são compromissos contínuos que requerem manutenção regular.

## Exemplos de Áreas:

- **Saúde e Fitness**: Exercícios, nutrição, bem-estar
- **Carreira**: Desenvolvimento profissional, habilidades
- **Finanças**: Orçamento, investimentos, planejamento financeiro
- **Relacionamentos**: Família, amigos, networking
- **Desenvolvimento Pessoal**: Aprendizado, hobbies, crescimento
- **Casa**: Manutenção, organização, projetos domésticos

## Como Usar:

1. Crie um arquivo `.md` para cada área importante da sua vida
2. Use como referência para acompanhar o estado de cada área
3. Revise regularmente durante suas revisões semanais
