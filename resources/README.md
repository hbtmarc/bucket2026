# Recursos

Esta pasta contém materiais de referência, pesquisas, artigos salvos, e outros conteúdos que você pode precisar consultar no futuro.

## Tipos de Recursos:

- **Aprendizado**: Cursos, tutoriais, guias
- **Inspiração**: Ideias, citações, exemplos
- **Referência**: Documentação, checklists, templates
- **Pesquisa**: Artigos, estudos, anotações

## Como Usar:

1. Organize por tema ou categoria
2. Adicione descrições breves para facilitar a busca
3. Revise periodicamente para manter relevante
4. Use links quando possível para evitar duplicação
