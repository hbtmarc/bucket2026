# Notas Diárias

Esta pasta contém suas notas diárias para capturar pensamentos, tarefas e reflexões do dia a dia.

## Como Usar:

1. Crie uma nova nota diária usando o template: `templates/daily-note-template.md`
2. Nomeie o arquivo com a data: `YYYY-MM-DD.md` (ex: `2026-01-21.md`)
3. Preencha ao longo do dia
4. Revise no final do dia

## Estrutura Sugerida:

```
daily-notes/
├── 2026-01/
│   ├── 2026-01-01.md
│   ├── 2026-01-02.md
│   └── ...
├── 2026-02/
│   └── ...
└── README.md
```

## Dicas:

- Seja consistente com o formato
- Não se preocupe com perfeição
- Capture ideias rapidamente
- Revise ao final da semana
