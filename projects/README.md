# Projetos

Esta pasta contém seus projetos ativos - objetivos com prazo definido e resultado específico.

## Características de um Projeto:

- Tem um objetivo claro
- Tem uma data de conclusão
- Requer múltiplas etapas
- Quando concluído, move para a pasta `archive/`

## Como Usar:

1. Crie uma pasta para cada projeto ativo
2. Use o template de projeto (`templates/project-template.md`)
3. Mantenha apenas projetos ativos aqui (máximo 5-7)
4. Mova projetos concluídos para `archive/`

## Exemplos de Projetos:

- Lançar novo produto
- Organizar evento
- Aprender nova habilidade
- Reforma da casa
