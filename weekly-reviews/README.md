# Revisões Semanais

Esta pasta contém suas revisões semanais para refletir sobre progressos, ajustar planos e preparar a próxima semana.

## Como Usar:

1. Reserve tempo no final de cada semana (domingo ou sexta)
2. Use o template: `templates/weekly-review-template.md`
3. Nomeie como: `YYYY-Semana-NN.md` (ex: `2026-Semana-03.md`)
4. Revise:
   - Notas diárias da semana
   - Progresso em projetos
   - Estado das áreas de vida

## Benefícios:

- Perspectiva clara do progresso
- Ajuste de prioridades
- Celebração de conquistas
- Planejamento proativo

## Checklist da Revisão:

- [ ] Revisar notas diárias
- [ ] Atualizar projetos ativos
- [ ] Verificar áreas de vida
- [ ] Planejar próxima semana
- [ ] Capturar aprendizados
